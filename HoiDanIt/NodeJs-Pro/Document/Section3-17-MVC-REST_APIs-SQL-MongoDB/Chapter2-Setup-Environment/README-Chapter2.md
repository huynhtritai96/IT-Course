# Chapter 2 - Setup Environment

This chapter covers the tooling setup required to follow the course. It focuses on keeping versions consistent to avoid environment-specific bugs.

## Goals
- Install the required tools and align versions with the course
- Configure the editor for consistent formatting
- Set up Git for source control
- Prepare a browser for testing web apps

## Lessons
1. #9. Tooling setup overview (mandatory)
2. #10. Install Node.js (course uses Node.js 22.13.0)
3. #11. Use Node.js with NVM (extra)
4. #12. Install Visual Studio Code
5. #13. Configure Visual Studio Code
6. #14. Why the course uses VSCode
7. #15. Install and use Git
8. #16. Install Google Chrome (English UI preferred)

## Key Notes From The Chapter
- Do not skip setup videos. Installation and configuration both matter.
- Use the same tool versions as the course to minimize unexpected bugs.
- If you only have one Node.js project, stick to a single Node version.
- Use NVM only when you truly need multiple Node versions.

## Reference Commands
Check Node.js version:
```bash
node -v
```
